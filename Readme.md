# Web-based word qlock using:

* [Svelte](http://svelte.technology/)
* [ES6 (via BabelJS)](http://babeljs.io/)
* [Webpack](http://webpack.github.io)

----

## Install & Run:

### Development:

    npm install
    npm start
    open http://localhost:3000

### Production:

    npm install
    webpack -p
    Host index.html and bundle.js wherever you want.
